# การติดตั้ง Python

## วิธีที่ 1: ติดตั้งจาก Microsoft Store
1. เปิด Microsoft Store
2. ค้นหา "Python"
3. ติดตั้ง Python 3.11 หรือใหม่กว่า

## วิธีที่ 2: ดาวน์โหลดจาก python.org
1. ไปที่ https://www.python.org/downloads/
2. ดาวน์โหลด Python เวอร์ชันล่าสุด
3. ติดตั้งและเลือก "Add Python to PATH"

## การรันโปรเจ็ค
หลังติดตั้ง Python แล้ว:
1. Double-click ไฟล์ `run.bat`
2. หรือเปิด Command Prompt และรัน:
   ```
   python app.py
   ```
