# Agentic AI Hackathon for SupplyUse

ระบบจัดการสต๊อกวัสดุสิ้นเปลืองสำหรับสถานีน้ำมัน

## คุณสมบัติ

1. **ข้อมูลหลักผลิตภัณฑ์** - จัดการข้อมูลสินค้าและวัสดุสิ้นเปลือง
2. **รับเข้าสต๊อก** - บันทึกการรับเข้าสินค้าด้วยการสแกนบาร์โค้ด
3. **เบิกออกสต๊อก** - บันทึกการเบิกจ่ายสินค้าด้วยการสแกนบาร์โค้ด
4. **รายงานความเคลื่อนไหวสต๊อก** - ดูรายงานและแจ้งเตือนสต๊อกต่ำ

## การติดตั้ง

1. ติดตั้ง dependencies:
```bash
pip install -r requirements.txt
```

2. รันแอปพลิเคชัน:
```bash
python app.py
```

3. เปิดเบราว์เซอร์ไปที่ `http://localhost:5000`

## การใช้งาน

### เพิ่มสินค้า
- ไปที่หน้า "ข้อมูลสินค้า" และคลิก "เพิ่มสินค้า"
- กรอกข้อมูลสินค้าหรือใช้การสแกนบาร์โค้ด

### รับเข้าสต๊อก
- ไปที่หน้า "รับเข้า"
- เลือกสินค้าหรือสแกนบาร์โค้ด
- ระบุจำนวนและข้อมูลอื่นๆ

### เบิกออกสต๊อก
- ไปที่หน้า "เบิกออก"
- เลือกสินค้าหรือสแกนบาร์โค้ด
- ระบุจำนวนที่ต้องการเบิก

### ดูรายงาน
- ไปที่หน้า "รายงาน" เพื่อดูความเคลื่อนไหวและแจ้งเตือนสต๊อกต่ำ

## เทคโนโลยีที่ใช้

- **Backend**: Flask (Python)
- **Database**: SQLite
- **Frontend**: Bootstrap 5, HTML, CSS, JavaScript
- **Barcode Scanner**: ZXing Library
